about=EIS (Exploration Information System) QGIS Plugin is a tool for prospectivity
    mapping of critical raw materials. EIS QGIS Plugin consists of two main components:
    EIS Wizard, a plugin application with guided processes and workflows and EIS
    Processing Algorithms, the individual tools and algorithms available in QGIS
    Processing Toolbox. To run EIS QGIS Plugin, you need to install EIS Toolkit in a
    Python environment and configure the environment in the plugin. More detailed
    instructions can be found in EIS QGIS Plugin GitHub repository
    (https://github.com/GispoCoding/eis_qgis_plugin) and EIS Toolkit GitHub repository
    (https://github.com/GispoCoding/eis_toolkit).
